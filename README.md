# 🌍 Bangladesh Climate Risk Prediction

An interactive Machine Learning application that predicts flood risk levels in Bangladesh based on key environmental and demographic factors.

---

## 🚀 Live App
[
(Add your Streamlit link here after deployment)](https://bangladesh-climate-risk-prediction-biykw56xkw42a8457fxy4a.streamlit.app/)

---

## 📊 Features

* Predict flood risk level (Low / Medium / High)
* User input for rainfall, population density, elevation, and temperature
* Machine Learning model using Random Forest
* Interactive interface built with Streamlit

---

## 🛠 Tools Used

* Python
* Streamlit
* Pandas
* Scikit-learn

---

## 📁 Project Structure

data/ → Dataset used for training
app.py → Main ML application
requirements.txt → Dependencies

---

## 📌 How It Works

* The model is trained on climate-related data
* Inputs are provided through sliders
* The system predicts flood risk using a trained ML model
* Results are displayed instantly

---

## 🎯 Purpose

This project demonstrates how Machine Learning can be applied to predict climate-related risks and support data-driven decision-making in vulnerable regions like Bangladesh.
